# Arman Singh — Portfolio

A professional portfolio website designed and developed for Arman Singh, a Full Stack Developer and Brand Strategist. This repository contains the source code for the portfolio, showcasing projects, skills, and experiences.

## 🚀 Live Demo

You can view the portfolio here once deployed. (Make sure to update this with your actual live URL)

## 📖 Overview

This portfolio is built to reflect a high-end, responsive, and interactive digital experience. It features a 3D glassmorphic design, smooth micro-animations, and dynamic interactions, optimized for both desktop and mobile devices.

### Sections Included:
- **Hero Section**: A dynamic introduction with 3D elements and floating chips.
- **About Me**: Professional summary highlighting experience in building digital businesses.
- **Skills Matrix**: Categorized technical and business skills.
- **Education**: Academic background and achievements.
- **Case Studies**: Selected projects including Nuclipp, JEOswap, and Trackademix.
- **Live Previews**: Interactive mobile-mockup interfaces for deployed projects.
- **Contact**: A dedicated section to get in touch.

## 🛠️ Built With

- **HTML5**: Semantic and accessible structure.
- **CSS3 (Vanilla)**: Custom styling, CSS Variables, Flexbox, Grid, and Animations.
- **JavaScript**: Interactive UI elements, dynamic modals, and smooth scrolling.
- **FontAwesome**: Scalable vector icons.

## ⚙️ Features

- **Responsive Design**: Fully responsive layout optimized for all screen sizes, featuring horizontal swipe-snapping on mobile.
- **Glassmorphism**: Modern frosted glass effect using CSS `backdrop-filter`.
- **Dynamic Interactions**: Reveal on scroll animations and interactive modal iframes for project previews.
- **Infinite Marquee**: Smooth, auto-scrolling brand partner showcase.
- **Performance Optimized**: Lightweight assets and clean code structure.

## 💻 Running Locally

To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/armansingh1196/My-Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd My-Portfolio
   ```
3. Open `index.html` in your preferred web browser. Alternatively, you can use a local development server like Live Server (VS Code extension) for automatic reloading.
